Основной сайт

Сайт преднозначен для использования шрифтов на вашем сайте.
На данный момент доступно 4 шрифта:

  • OpenSans
  • NotoSansJP
  • Roboto
  • SF Pro Display(При использовании шрифта вы соглашаетесь с пользовательским соглашением Apple.)

Как подключить наши шрифты?

!!! Шрифты не принадлежат конкретно нам, мы только даем возможность их подключить.

1. Создайте css файл в котором будет подключаться шрифты к сайту, или используйте уже существуюший css фай.

2. Введите в css файл данный код:

@import url(' https://fontsprobrovovayt.github.io/fonts/Roboto/style.css ');
/* Подключить шрифт Roboto к css файлу */

@import url(' https://fontsprobrovovayt.github.io/fonts/OpenSans/style.css ');
/* Подключить шрифт OpenSans к css файлу */

@import url(' https://fontsprobrovovayt.github.io/fonts/NotoSansJP/styles.css ');
/* Подключить шрифт NotoSansJP к css файлу */

@import url(' https://fontsprobrovovayt.github.io/fonts/SFProDisplay/styles.css ');
/* Подключить шрифт NotoSansJP к css файлу */

* {
font-family: 'Roboto', sans-serif;
/* Подключить шрифт Roboto */

font-family: 'OpenSans', sans-serif;
/* Подключить шрифт OpenSans */

font-family: 'NotoSansJP', sans-serif;
/* Подключить шрифт NotoSansJP */
font-family: 'SF Pro Display', sans-serif;
/* Подключить шрифт SF Pro Display */
}

3. На вашем сайте должены появиться шрифты.
